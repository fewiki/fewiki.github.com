# 在项目中使用 we-ui
## 安装
```
npm install we-ui
// 如果安装较慢，可使用
// 淘宝镜像： cnpm install we-ui --registry=https://registry.npm.taobao.org
// 或yarn安装： yarn install we-ui
```

## 引入
在vue入口引入，如main.js中
```
import WeUi from 'we-ui'
Vue.use(WeUi)
```
